# Taro Stack

- Framework: Taro + React + TypeScript as first-choice cross-platform stack.
- Target mapping: `taro build --type weapp`, `taro build --type tt`, `taro build --type alipay`.
- Use `@tarojs/components` primitives for shared UI shells.
- Keep page and component source under `src/` and avoid platform-native folders in shared flow.
