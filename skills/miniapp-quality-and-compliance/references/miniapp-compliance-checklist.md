# 小程序合规检查清单

## 总控清单

- [ ] 是否存在硬编码 `appid`、密钥、签名字符串？
- [ ] `wx` / `tt` / `my` 调用是否全部经过 adapter boundary？
- [ ] 敏感场景是否有权限分步提示与失败回退？
- [ ] 关键隐私字段是否在采集、展示、持久化中被限制范围？
- [ ] 是否有 `release blocking` 风险说明和处置建议？

## 风险分级

- **P0**: 严重权限/隐私风险、支付状态错误闭环、数据泄露风险 → `needs_human` / blocked。
- **P1**: 生命周期或错误处理断裂 → 明确修复项，必要时降级为 `needs_human`。
- **P2**: 可观测但不阻塞项 → 写进 release note 与自测记录。
